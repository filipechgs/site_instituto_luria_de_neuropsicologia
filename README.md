# Proposta de Desenvolvimento do Site do Instituto Luria de Neuropsicologia
Site desenvolvido para o Instituto Luria de Neuropsicologia.

## Requisitos solicitados

### Sessão Institucional:

- Apresentação da clínica e sua missão.
- Descrição das atividades oferecidas (atendimento, supervisão, cursos, práticas previsionadas).
- Apresentação da equipe (diretores, sócios e profissionais associados).

### Sessão de Contato:

- Endereço
- Telefone
- E-mail 
- Links para as redes sociáis
- Link direto para contato via WhatsApp.

### Página de Cursos:

Links de redirecionamento para os cursos hospedados na Hotmart.

### Links para as Redes Sociais:

- Link para o canal do YouTube (em processo de reativação).
- Link para o Instagram do Instituto.
- Link para o Facebook do Instituto.

### Recursos Adicionais

- Blog / Área de Reflexões e Materiais:
  - Será criado pelos clientes via CMS
- Área de Downloads:
  - Contem link para a página do artigo publicado   

### Publicações e Eventos:
Página para disponibilização de materiais como e-books, cartilhas e artigos.

- Listagem de publicações do Instituto (artigos, papers, estudos).
- Links para os artigos publicados.
- Divulgação de participações em eventos (incluindo fotos e descrições de trabalhos apresentados).

### Funcionalidades Técnicas
- Design responsivo para acesso via desktop e mobile.
- Integração com WhatsApp e redes sociais.
